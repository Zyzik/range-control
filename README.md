# Range control

## Usage
$('.class').rangeControl({settings})
$('.class').data('range-control').method()

## Settings
* min (data-min)
* max (data-max)
* leftValue (data-left-value)
* rightValue (data-right-value)
* timeout (data-value-timeout)
* formatControlCallback - function

## Methods
* leftValue()
* rightValue()
* value()
* min()
* max()
* rebuild()
* render()
* destroy()
