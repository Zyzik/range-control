# Range control

## Usage
$('.class').rangeControl({settings})
$('.class').data('range-control').method()

## Settings
* startValue (data-start-value)
* endValue (data-end-value)
* valueStep (data-value-step)
* timeout (data-value-timeout)
* formatControlCallback - function

## Methods
* leftValue()
* rightValue()
* value()
